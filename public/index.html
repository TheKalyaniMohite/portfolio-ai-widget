EV Charging: Utilization & Reliability in California

GitHub Repository Live App
Built a reproducible pipeline and dashboards that map where public chargers exist relative to EV demand. Using AFDC station data + CA EV registrations (with a HUD ZIP→County crosswalk), I cleaned and modeled stations/ports, computed coverage (ports per 1,000 EVs) and DCFC share, and surfaced two action tools: a transparent Siting Score (low coverage × high EVs) and a Likely Busy station watchlist.
